# AI Services\n\n## OpenAI Integration\n\n### Models Used\n- **GPT-4 Turbo**: Content generation\n- **text-embedding-3-small**: Semantic search embeddings\n\n## Features\n\n### Content Generation\n\n#### Generate Articles\n```\nPOST /api/ai/generate\n{\n  \"type\": \"article\",\n  \"topic\": \"AI in education\",\n  \"tone\": \"professional\",\n  \"length\": \"long\"\n}\n```\n\n#### Generate Social Media Posts\n```\nPOST /api/ai/generate\n{\n  \"type\": \"social\",\n  \"platform\": \"twitter\",\n  \"topic\": \"AI trends\"\n}\n```\n\n#### Generate Titles & Hooks\n```\nPOST /api/ai/titles\n{\n  \"topic\": \"Machine Learning\",\n  \"count\": 10\n}\n```\n\n### Text Analysis\n\n#### Improve Grammar\n```\nPOST /api/ai/improve\n{\n  \"text\": \"Your content here\"\n}\n```\n\n#### Summarize Content\n```\nPOST /api/ai/summarize\n{\n  \"text\": \"Long article\",\n  \"maxLength\": 200\n}\n```\n\n### Knowledge Bot\n\n#### Ask Question\n```\nPOST /api/ai/ask\n{\n  \"query\": \"What did I write about AI?\",\n  \"limit\": 5\n}\n```\n\nReturns relevant content with sources and confidence scores.\n\n## Vector Search\n\n- Powered by Supabase pgvector\n- Cosine similarity for semantic matching\n- Threshold: 0.7 for high relevance\n\n## Prompt Engineering\n\nCustomizable system prompts in `/ai/prompts/`\n\nEach content type has optimized prompts for:\n- Tone and style\n- Structure and formatting\n- SEO optimization\n- Brand voice consistency
